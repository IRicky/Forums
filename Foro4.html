Ambos modelos, tanto el vectorial como el raster, son útiles para representar datos geográficos, pero uno puede ser más apropiado que otro cuando se trata de representar un tipo de datos geográficos en particular.

En general, el modelo de datos vectorial es útil para representar entidades que tienen límites discretos, mientras que el modelo de datos raster es más útil para representar datos geográficos continuos—fenómenos como la precipitación, la elevación o la temperatura— que no tienen límites bien definidos y que normalmente cambian gradualmente a lo largo del área determinada.
